pub fn Ok<T>(t: T) -> Result<T>
Expand description

Equivalent to Ok::<_, eyre::Error>(value).

This simplifies creation of an eyre::Result in places where type inference cannot deduce the E type of the result — without needing to write Ok::<_, eyre::Error>(value).

One might think that eyre::Result::Ok(value) would work in such cases but it does not.
